# Pre-Commit Configuration (Bastion)

This project uses [pre-commit](https://pre-commit.com/) to automate linting and validation checks.

## Setup

Install pre-commit if not already installed:
```bash
pip install pre-commit
```

Install the git hooks:
```bash
pre-commit install
```

## Available Hooks

- **validate-doc-links**: Checks for broken links in Markdown documentation files.

## Running Hooks

Run on staged files (default):
```bash
pre-commit run
```

Run on all files:
```bash
pre-commit run --all-files
```

Run a specific hook:
```bash
pre-commit run validate-doc-links --all-files
```

Bypass hooks (not recommended):
```bash
git commit --no-verify
```
