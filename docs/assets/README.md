# Brand Assets

Generate icons and a social preview image from the SVG logo.

```bash
chmod +x scripts/generate-brand-assets.sh
scripts/generate-brand-assets.sh
```

Outputs are written to [logos](./logos):
- social-preview.png — upload in GitHub → Settings → General → Social preview
- favicon-16.png, favicon-32.png, apple-touch-icon.png
- icon-512.png
